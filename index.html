Pomodoro Timer

Go to project

Technologies

Javascript, SASS, CSS, HTML, Web Audio API, Web Notification API, Web Storage API, Gulp

Description

This project was part of the FreeCodeCamp curriculum. Its objective is to implement the Pomodoro Technique.

I've used in this project everything that I haven't had the opportunity to work with 'til now (and could fit in the project, of course), but always made me curious. Audio, notifications and storage APIs.

With Audio API, I've made just a simple mp3 beep play. This was the first step to a more complex project that I've made that is the Simon Game.

Storage API made possible storing data, like the history of pomos made. It's a simple yet powerful tool to temporarily persist data in the browser, without the need of a server. I hope to use this soon in some offline first applications.

Another simple and powerful tool is the Notifications API. By using it, the application can communicate with the user without him being looking at the page. That's what this app does when a pomo is finished.

The idea for the UI came from here. A nice, simple and visual way to show the timer progress.

FreeCodeCamp News

Go to project

Technologies

Javascript, AngularJS, SASS, CSS3, HTML, moment.js, Gulp

Description

This project was part of the FreeCodeCamp curriculum. Its objective is to receive data from the FreeCodeCamp's News API and display it in a customized way.

The biggest point in this project was the use of flexbox and CSS multiple column layout. The last isn't well known and helps the layout of interfaces like Pinterest (masonry grid), with native CSS that can be easily made responsive. It automagically distributes content equally between columns. Together, these technologies support every major browser nowadays (not IE8/9).

AngularJS was used for it's simple templates, AJAX requests and usage. Moment.js is a simple library to parse and display dates.
