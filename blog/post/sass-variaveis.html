SASS: COMO USAR VARIÁVEIS

Essa postagem do blog vai lhe permitir sair usando o SASS em seus projetos e diminuir uma quantidade significativa de código. E olha que variáveis são o elemento mais básico no SASS.

Pense nas variáveis do SASS da mesma forma como numa linguagem de programação. Elas são uma forma de você guardar a informação em um local para reutilizar em algum momento. No nosso caso, usaremos essas variáveis para armazenar cores, fontes, medidas e quaisquers outros valores de CSS que você imaginar e que será reutilizado.

Para criar uma variável no SASS, usamos o símbolo $. Veja um exemplo: 

$fonte:    'Open Sans', sans-serif;
$cor-primaria: #333;

body {
  font: 100% $fonte;
  color: $cor-primaria;
}

Quando o SASS for processado, ele vai substituir cada variável pelo valor a que ela corresponde. Isso é incrivelmente útil se levarmos em conta um projeto que precisaríamos manter a identidade de cor da marca consistente por todo site.

body {
  font: 100% 'Open Sans', sans-serif;
  color: #333;
}
